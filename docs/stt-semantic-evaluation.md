# Voxtral Mini 3B STT 意味的精度再評価

- **日時**: 2026-02-13
- **対象**: stt-bench-results.md（ベストプロンプト C_english での50文テスト）
- **目的**: 完全一致ではなく「意味が通じるか」で再評価

## 評価基準

| 記号 | 判定 | 説明 |
|------|------|------|
| ✅ | 意味一致 | 意味が同じ（言い換え・表記違い程度） |
| ⚠️ | 部分一致 | 大意は合ってるが一部情報欠落・誤り。会話として通じる |
| ❌ | 意味不一致 | 意味が変わってる、ハルシネーション、全く別の内容 |

| カテゴリ | ✅意味一致 | ⚠️部分一致 | ❌意味不一致 | 意味的精度(✅+⚠️) | 完全一致精度(元) |

| 数字・固有名詞 | 3 (30%) | 2 (20%) | 5 (50%) | **50%** | 40% |

| 長文 | 3 (30%) | 6 (60%) | 1 (10%) | **90%** | 60% |
| **全体** | **23 (46%)** | **14 (28%)** | **13 (26%)** | **74%** | **48%** |

### 精度比較サマリー

| 指標 | 値 |
|------|------|
| 完全一致精度（元評価） | 48% |
| 意味的精度（✅+⚠️） | **74%** |
| 厳密意味一致（✅のみ） | 46% |
| 意味不一致率（❌） | 26% |

### 分析

1. **意味的精度74%は完全一致48%から+26ポイント改善** — 「意味が通じる」基準では大幅にスコアが上がる
2. **日常会話と長文が90%** — 言い換えが多いが意味は通じる。LLMの「意訳」特性がここではプラスに働く
3. **技術用語・質問文が70%** — 完全一致では33-37%だったが、WebSocket→ウェブソケット等の表記揺れを許容すると大幅改善
4. **数字・固有名詞は50%で最低** — 数値・地名の誤認識は意味が変わるため意味的評価でも改善幅が小さい
5. **26%が意味不一致** — ハルシネーション・崩壊が全カテゴリで発生。実用にはポストフィルタリングが必要

### 結論

Voxtral-Mini-3Bは完全一致48%だが、意味的精度では74%。
日常会話や長文では実用レベル（90%）に近いが、数字・固有名詞の認識が弱点（50%）。
音声アシスタント用途（大意が伝わればOK）なら検討の余地はあるが、
正確な数値や固有名詞が必要な用途にはWhisper等の専用モデルが必要。
